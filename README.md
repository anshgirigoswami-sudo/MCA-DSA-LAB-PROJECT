# MCA-DSA-LAB-PROJECT

This repository contains Data Structure and Algorithm lab assignments for MCA coursework.

## Assignment Overview

The project includes Python programs organized assignment-wise inside the DATA STRUCTURE folder.

### Files Included

- DATA STRUCTURE/Lab_assignment01.py
- DATA STRUCTURE/Lab_assignment02.py
- DATA STRUCTURE/Lab_assignment03.py
- DATA STRUCTURE/Lab_assignment04.py
- DATA STRUCTURE/Lab_assignment05.py

## Objective

These assignments are intended to practice core DSA concepts such as:

- Arrays and basic operations
- Searching and sorting logic
- Stacks, queues, and linked structures
- Problem-solving with algorithmic thinking

## How to Run

1. Install Python 3.x.
2. Open the project in VS Code or terminal.
3. Run any assignment file:

	python "DATA STRUCTURE/Lab_assignment01.py"

Replace the filename with the assignment you want to execute.

## Note

Each file is a separate lab task and can be executed independently.
